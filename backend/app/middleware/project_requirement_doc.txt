Barber Shop Management System

A slot-based booking system that connects users and barbers to reduce uncertainty, waiting time, and idle hours.

System Design
1) USER: 1) Barber Shop Owner, 2) Customer

2) FEATURES: 
User can see the idle slots
Rating and services given to the salon/ barber
Provide interface for multiple services (like haircut, makeup, spa)
Adv. slot booking in the app 
Payment gateway (Razorpay)
Recommendation according to user requirement
Provide a global search bar with advanced filtering (keyword (location) + long + lat)
Add Extra Points feature  [using 100 points haircut free]
Users doing home services can also register

CRUD Operation for barber shop and profile 
CRUD  Operation for User profile

3) COMPONENTS
